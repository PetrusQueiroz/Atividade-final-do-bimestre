Site sobre o CEEP

Sobre o projeto

Esse projeto foi desenvolvido com o objetivo de criar um site sobre o CEEP (Centro Estadual de Educação Profissional), mostrando informações como a história da escola, cursos, estrutura e também uma página de matrícula.

A ideia foi colocar em prática os conhecimentos de HTML e CSS vistos em sala.

Objetivo

O objetivo principal do site é apresentar a escola de forma organizada e clara, além de cumprir os requisitos da atividade proposta pelo professor.
